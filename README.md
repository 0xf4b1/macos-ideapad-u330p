# macos-ideapad-u330p

macOS Big Sur Open Core setup configured for Lenovo IdeaPad u330p.

Tested setup:

- CPU: Intel(R) Core(TM) i5-4200U
- GPU: Intel HD Graphics 4400
- WiFi: Intel Wireless-N 7260

## Issues

- WiFi does not work after sleep
- Bluetooth
- Brightness Control
- Battery Indicator
- Card Reader
- Webcam
